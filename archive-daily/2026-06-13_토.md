    <span class="ni__n">06</span>
    <a class="ni__t" href="https://blackforestlabs.ai/" target="_blank" rel="noopener">FLUX (Black Forest Labs) — 오픈소스 이미지 모델 <span class="ni__arrow">↗</span></a>
    <span class="ni__tr"><span class="ni__src">도감</span><span class="ni__score">6/10</span></span>
  </header>
  <p class="ni__body">FLUX는 오픈 웨이트(open weights)로 공개돼 로컬 GPU에서 돌릴 수 있는 이미지 생성 모델입니다(diffusion = 노이즈에서 점점 그림을 또렷하게 만드는 방식). ControlNet(외곽선·구도를 고정해 생성 방향을 잡아주는 보조망)과 묶으면 내 매스/스케치의 구도를 유지한 채 재질·분위기를 입히는 렌더 보정에 닿습니다. 공개 가중치 변형을 쓰면 로컬 추론 비용을 0으로 둘 수 있어 클라우드 렌더 도구의 대안입니다.</p>
  <ul class="ni__pts"><li>오픈소스(무료) · 로컬 GPU 필요(VRAM 부담)</li><li>스케치+구도 → 재질·분위기 입힌 이미지</li><li>셋업 난이도 있음 — ComfyUI 등 파이프라인 구성 필요</li></ul>
  <div class="ni__meta">
    <div class="ni__row"><dt>한줄평</dt><dd>GPU 있고 손품 들일 의향이 있으면 렌더 보정 비용을 0으로 내림. 셋업 장벽이 있어 누구에게나 쉬운 건 아님 — 로컬파 실무자용.</dd></div>
    <div class="ni__row ni__row--do"><dt>써먹기</dt><dd><p class="ni__do-text">라이노 클레이 뷰를 ControlNet 구도로 고정하고 FLUX로 재질·하늘을 입혀 패널 배경 렌더로. 로컬이라 반복 생성 비용 없이 시안을 많이 뽑을 때.</p></dd></div>
    <div class="ni__row ni__row--why"><dt>성숙도</dt><dd>공개된 오픈 웨이트 모델이라 로컬 구동은 무료. 단 API·엔터프라이즈는 유료 트랙도 병행. 클라우드 렌더 구독이 부담될 때 자가 파이프라인을 갖추기 좋은 시점입니다.</dd></div>
